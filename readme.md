3. Novo Foco: Chamada e Atas:


Problema: tempo perdido, erros e possibilidade de fraude.
Solução: registro imutável com timestamp e assinatura digital.
Diferencial: prova criptográfica de presença.


Primeiros passos:

1. só será possível registrar uma única vez, então muito cuidado nessa parte

2. o sistema ficará fechado até o professor autorizar. Um qrcode será criado e pronto para os registros

3. a blockchain só vai armazenar dados dos alunos respectivamente e será aberto ao público, caso tenha alguma confusão na hora da chamada, o aluno poderá verificar na blockchain.

4. sera praticamente nulo as chances de manipulações e fraudes.
